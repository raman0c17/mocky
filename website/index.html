Get started

Up and running in a minute

Install from PyPI, or clone and develop locally in a virtual environment.

Install & run

# install
pip install mocky

# launch the interactive CLI
mocky

Use as a library

from mocky import (
    TemplateLibrary,
    MarkdownParser,
    PowerPointGenerator,
)

md = TemplateLibrary.build_markdown(
    "Launching Mocky 1.0", "product-launch")
parsed = MarkdownParser().parse_markdown(
    TemplateLibrary.save_markdown(
        md, "generated_markdowns", "launch.md"))
PowerPointGenerator().create_presentation(
    parsed["slides"], "presentations/launch.pptx")
